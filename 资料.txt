1，PID计算公式有几种？
--3种，标准，位置，增量

2，位置式PID为什么称位置？
--在标准PID基础上，把积分项和微分项改为了位置形式。
--位置型PID公式将积分项和微分项改为了对误差的位置变化进行运算，输出的控制量也是与位置变化量相关的。
--可以将位置型PID公式看作是从连续时间控制系统到离散时间控制系统的变化。

3，电流环在控制上有什么应用？
--流环通常是内环，速度或位置环通常是外环。
--电流环PID将直接控制电机的电流输出，以实现所需的电机转矩。速度环或位置环PID将根据电机的实际运动情况对电流环的输出进行反馈调节，以控制电机的运动速度或位置。
--这种串联结构还可以减小控制系统的稳态误差，从而使控制系统更加准确

4，mpu6050
--MPU6050集成了三轴加速度计和三轴陀螺仪，可以同时测量和输出物体的加速度和角速度信息。MPU6050还具有温度传感器和数字信号处理器，能够对传感器数据进行处理和滤波，提高数据的准确性和稳定性。MPU6050采用I2C总线协议进行通信，具有多种测量范围和采样率可选。
--加速度计是一种可以测量物体在某一方向上加速度的传感器，通常使用MEMS技术制造，其原理是利用微小的质量块、弹簧和电容等结构实现加速度的测量。陀螺仪则是一种可以测量物体在某一方向上角速度的传感器，其原理是利用微小的旋转结构实现角速度的测量。

5，在外设初始化后，延时一段时间出现程序卡住情况，原因是？
--采用软件循环的方式实现延时函数，即通过循环来等待一段时间，那么延时1ms就需要进行非常多的循环，如果循环次数过多，就可能会让程序卡住，导致后续任务无法正常执行。

6，介绍IIC
--IIC（Inter-Integrated Circuit）又称I2C，是一种串行通信总线，由飞利浦公司（Philips）于20世纪80年代开发出来的。I2C总线是一个双向、2线制的串行总线，主要用于在微控制器和外围设备之间传输数据。
--I2C总线由两条线组成：SDA（数据线）和SCL（时钟线）。SDA线传输数据，SCL线传输时钟信号。它支持多个设备（最多128个设备）连接在同一条总线上，并且可以在同一时间由一个主设备（Master）控制通信，或者多个从设备（Slave）进行数据传输。
--I2C总线的优点是它具有较少的引脚数、传输速度适中、传输距离短、总线通讯协议简单易懂等特点，因此广泛应用于许多嵌入式系统和数字电路中，比如单片机、传感器、LCD屏幕、数字电路、存储器等设备之间的通讯。
--在使用I2C总线进行通讯时，需要在主设备和从设备之间传输一些控制信息，比如设备地址、读写控制、数据内容等。主设备通过发送控制信息来读取或写入从设备的数据，而从设备则在接收到控制信息后将数据发送回主设备。通讯协议非常灵活，可以实现从设备之间的数据传输，也可以实现主设备和从设备之间的数据传输。

8，全局变量如果不初始化，会赋什么值?
--如果全局变量没有显式地初始化，它将根据其数据类型自动初始化为其所属类型的默认值。
--常见数据类型的默认值：
--int, float, double, long等数值类型默认值为0。
--char类型默认值为空字符('\0')。
--指针类型的默认值为NULL，即指针不指向任何地址。
--bool类型默认值为false。
--结构体和类类型的默认值为其成员变量的默认值，即这些成员变量的类型的默认值。
--需要注意的是，局部变量在定义时不会自动初始化，它们的值是未知的。在使用这些变量之前，必须对它们进行明确的初始化。

9，数字0和字符0的ASCII码?
--数字 0 的 ASCII 码是 48（0x30），而字符 0 的 ASCII 码也是 48（0x30），它们的 ASCII 码是相同的。这是因为在 ASCII 码表中，数字 0 到 9 和字符 0 到 9 的编码是连续的，都是从 48 开始依次增加的。

10，只有一个任务要执行，任务每次的执行时间是多少？
--可以通过以下方式测出
void vTaskFunction(void *pvParameters) {
  TickType_t xStartTime, xEndTime;
  while(1) {
    xStartTime = xTaskGetTickCount();
    // 执行任务操作
    xEndTime = xTaskGetTickCount();
    printf("Task execution time: %u ms\n", (xEndTime - xStartTime) * portTICK_PERIOD_MS);
  }
}
--任务执行程序中不能加vTaskDelay（），加了之后，任务会被挂起，无法测量时间。
